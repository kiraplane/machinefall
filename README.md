# Machinefall Wiki

Independent English guide hub for **Machinefall** at
`https://www.machinefall.wiki`.

The launch site covers:

- a stable first-campaign route;
- depots, logistics, workers, carts and naval transport;
- walls, towers, army formations, lairs and boss preparation;
- cities, refugee camps and technology-era planning;
- official Windows requirements, a no-score review and safe Steam downloads;
- Article, BreadcrumbList, ItemList, WebSite, Organization, VideoGame and VideoObject schema.

## Development

```bash
pnpm install
pnpm dev
pnpm lint
pnpm exec tsc --noEmit
pnpm next:build
pnpm build
```

## Source policy

- Current Steam release facts and developer answers outrank old Demo devlogs.
- Community performance claims remain labeled as reports unless confirmed.
- No fake codes, unsupported tier lists, cracks, APK mirrors, injectors or
  extracted game data.

## 直接竞品结论

- 专属 Wiki：未发现成熟站点。
- Articles：Steam discussions 有真实问题，但信息分散且旧版混杂。
- Data：官方资料足够构建首发攻略，尚不足以建立可靠单位数值数据库。
- Tools：未发现 calculator、tracker 或 interactive map；当前也没有足够稳定数据支撑自建。
- Video：官方频道有 trailer 与长实机，社区有 review；尚无系统化正式版攻略系列。
- Opportunity：首页 hub + 十篇问题型攻略 + 三篇购买/运行页面即可覆盖首发高意图，不应扩张到虚构的 30–50 页面。

## 版本冲突规则

- 正式版 Steam News > 当前开发者讨论回复 > Steam Store > 旧 itch devlog > 社区猜测。
- old devlog 中与当前公告冲突的物流限制、路线行为或 Demo 内容不进入正文。
- Stomper 只属于旧 Demo 语境；没有官方证据时不作为正式版最终 boss 名称。
- multiplayer 与 official mod support 当前都是明确的“没有”；不把社区注入方案转成推荐内容。
